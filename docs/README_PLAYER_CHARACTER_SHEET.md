# Player Character Sheet Guide

This guide explains how to use the player-facing character sheet in the TaleSpire 5e Toolset.

## Overview

The Player Character Sheet is designed to keep most of your active play information in one place.

Main tabs:
- Player Stats
- Actions
- Spell List
- Inventory
- Features & Traits
- Notes
- Extras
- Init List

The sheet also includes:
- Quick stat controls in the header
- Theme and settings controls
- A Homebrew menu for custom content and data import/export

## Header And Quick Stats

The top section of the sheet is your fast-access character area.

It includes:
- Initiative
- AC
- Speed
- Level
- XP
- Death saves
- Class display
- Current HP and max HP
- Temp HP
- Rest buttons
- Conditions
- Inspiration

### Link Mini

The `Link mini` button lets you connect an image from a TaleSpire mini to the sheet.

Current note:
- This works with TaleSpire base minis
- It does not fully support community content minis

### Conditions

Use the condition dropdown to add active conditions to your character.

Conditions can affect multiple parts of the sheet. Examples include:
- AC changes
- Damage bonuses
- Roll adjustments
- Other mechanical updates linked to the condition system

## Player Stats Tab

The Player Stats tab is your core character summary.

It includes:
- Ability scores
- Saving throws
- Skill modifiers
- Passive values
- Proficiency controls

### Editing Stats

Much of the sheet is editable, which allows you to:
- Enter manual values
- Fine-tune house rules
- Adjust a sheet that does not exactly match a standard build path

Example:
- You can change a skill to use a different stat if your table supports that as a house rule, such as Strength-based Intimidation.

## Actions Tab

The Actions tab is designed for fast play at the table.

Use it to organize:
- Attacks
- Actions
- Bonus actions
- Reactions
- Other actions

### What This Section Does

This tab helps you:
- Keep combat options readable
- Reduce time spent searching through notes
- Organize actions by type

### Action Sources

Actions can come from:
- Equipped items
- Character setup
- Manual entries
- Features and traits

### Add Or Edit Actions

You can manually add actions using the `+` control in the actions area.

You can also:
- Filter by action category
- Edit action details
- Open action settings using the `i` button

Important note:
- Traits, conditions, and equipment can influence how actions behave, so make sure the action type and settings are correct.

## Spell List Tab

The Spell List tab organizes your cantrips and leveled spells into a readable casting interface.

You can use it to:
- Review spell details
- Track spell slots
- Add spells
- Adjust spell slot layouts

### Spell Details

Click the `i` icon on a spell to open its details.

### Adding Spells

You can add spells by:
1. Going to the level section where the spell belongs
2. Clicking the `+` control
3. Typing the spell name into the search field
4. Selecting the spell you want

### Upcast Display

If upcast display is enabled in settings, the sheet can show more upcast information for relevant spells.

You can turn this behavior on or off from the settings menu.

## Inventory Tab

The Inventory tab separates your items into categories such as:
- Equipment
- Backpack
- Other possessions
- Attunement

It also helps track:
- Item details
- Weight
- How equipment affects the rest of the sheet

### Why Inventory Matters

Inventory is not just a list. Items can affect your character sheet mechanically.

Examples:
- Magic items can change AC
- Magic items can change saves
- Equipped gear can affect actions
- Attuned items can apply ongoing bonuses

## Features & Traits Tab

The Features & Traits tab is where you manage class features, species traits, feat-style effects, and homebrew mechanics.

This section is especially important for characters with:
- Custom mechanics
- Ongoing bonuses
- Limited-use features
- Triggered adjustments

### What You Can Use It For

You can use this tab to manage effects such as:
- Fighting styles
- Special bonuses
- Homebrew rules
- Resource tracking
- Sheet-wide modifiers

This section can influence many other parts of the sheet, including:
- Actions
- Spell save DC
- Attack bonuses
- Damage bonuses
- Other calculated values

## Notes Tab

The Notes tab gives players grouped note sections inside the character sheet.

Good uses include:
- Quest notes
- Roleplay reminders
- NPC names
- Goals
- Item clues

If you like keeping everything in TaleSpire during a session, this is the best place for that.

## Extras Tab

The Extras tab is meant for alternate stat packages and companion-style tracking.

Use it for:
- Pets
- Wild shape forms
- Polymorph forms
- Summons
- Other alternate blocks

This helps keep those forms available without crowding the main sheet.

It can also work with homebrew monsters the player creates.

## Init List Tab

The Init List tab shows the player-facing initiative order during combat.

This helps players:
- See turn order
- Know who is up next
- Stay engaged between turns

The DM can also control which enemies are visible in the initiative view, so hidden enemies can still remain hidden when needed.

## Settings

The player settings menu includes:
- Language selection
- Theme controls
- Exhaustion homebrew toggle
- D&D version toggle
- Upcast display toggle

These settings let you shape the interface to fit your table and your character setup.

## Suggested Player Workflow

A good player workflow looks like this:

1. Import or load your character.
2. Review the quick stats in the header.
3. Check conditions, inspiration, and rest controls.
4. Make sure your actions, spells, and inventory are set up.
5. Use Features & Traits for anything that affects calculations.
6. Use Extras for alternate forms and companions.

## Tips

- If a number on the sheet seems off, check your conditions, traits, and equipped items first.
- Use the Actions tab for table-speed and the Features tab for background mechanics.
- Keep notes short and useful for active play.
- If your table uses house rules, the editable parts of the sheet give you room to support them.
