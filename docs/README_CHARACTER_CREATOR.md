# Character Creator Guide

This guide explains how to build a character with the Character Creator.

## Overview

The Character Creator is a guided build tool that walks through a character step by step.

Main tabs:
- Basics
- Species
- Class
- Abilities
- Backgrounds
- Summary

The Character Creator is best used as a build-and-export tool. After finishing a character, you can move that data into the main player sheet through the import workflow.

## Basics Tab

The Basics tab is where you define the core setup of the character.

This includes:
- Character name
- Alignment
- Hit point calculation method
- Ability score generation method

### Hit Point Method

You can choose how hit points are handled for the build.

### Ability Score Method

You can choose from:
- Manual or rolled
- Standard array
- Point buy

Choose the method that matches your table's character creation rules.

## Species Tab

The Species tab is where you select your character's species and related choices.

Use this section to:
- Read species details
- Choose a species
- Review traits
- Make species-related selections where available

Depending on the species, this can include:
- Ability bonuses
- Languages
- Traits
- Special options

## Class Tab

The Class tab is where you choose a class and handle class progression choices.

This section supports:
- Class selection
- Level progression
- Subclass selection
- Class features
- Class-specific choice handling

If your class has important build choices, this is one of the most important sections in the creator.

## Abilities Tab

The Abilities tab is where you assign and review your ability scores.

The exact interface changes depending on the ability-score method selected in Basics.

### Manual Or Rolled

Use this if:
- You already know your numbers
- Your table rolls for stats
- You want direct control

### Standard Array

Use this if your table uses the standard array method.

### Point Buy

Use this if your table builds characters with a point-buy system.

## Backgrounds Tab

The Backgrounds tab handles background-driven parts of character creation.

Use it to choose:
- Background identity
- Background-related proficiencies
- Flavor options
- Related feature choices where available

This helps round out the non-class side of the character.

## Summary Tab

The Summary tab is where you review the finished build.

Use it to:
- Check the overall result
- Confirm your selections
- Export the character data

## Saving And Using The Character

The Character Creator currently works best as a generation tool that produces character data for the main player sheet.

Typical workflow:

1. Build the character in the Character Creator.
2. Finish on the Summary tab.
3. Use the save action to copy the generated character data.
4. Open the Player Character Sheet.
5. Use the import workflow to bring the character into the main sheet.

## Best Practices

- Decide your hit point method before building too far.
- Decide your ability-score method first so the rest of the build stays consistent.
- Review class and species choices carefully before exporting.
- Use the Player Character Sheet after import for actual session play and long-term updates.

## Good Use Cases

The Character Creator is especially helpful when:
- Starting a brand-new campaign
- Building a new player character
- Rebuilding a paper character into the toolset
- Preparing a clean export for import into the player sheet
