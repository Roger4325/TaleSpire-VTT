# DM Screen Guide

This guide explains how to use the DM side of the TaleSpire 5e Toolset.

## Overview

The DM Screen is the main control surface for running sessions. It combines initiative management, quick-reference tables, spell lookup, notes, and homebrew tools in one place.

Main tabs:
- Initiative Tracker
- DM Tables
- Checklists
- Spell List
- Notes
- Google Docs

The header settings menu also gives you access to:
- Language selection
- Theme controls
- Exhaustion homebrew toggle
- D&D version toggle
- Homebrew tools

## Initiative Tracker

The Initiative Tracker is where most combat management happens.

### Add Monsters

1. Click the monster input at the top of the tracker.
2. Type part of the monster name.
3. Select the monster from the dropdown list.
4. A monster card will be added to the tracker.

### Add Players

1. Click the player input at the top of the tracker.
2. Choose a player from the connected player list.
3. The player will be added as a player card in the initiative tracker.

If a player has their character sheet open and synced, their information can update on the DM side when requested.

### Start Combat

Use these controls once creatures are added:
- `Auto Roll` to roll initiative for the tracker
- `Previous Turn` and `Next Turn` to move through combat
- The round counter to keep track of combat rounds

Initiative can also be edited manually by clicking into the initiative field on a card.

### Conditions and Roll State

At the top of the tracker you can:
- Apply conditions from the condition dropdown
- Switch the global roll mode between advantage, flat, and disadvantage

This is useful for quickly reflecting table-wide combat state without leaving the tracker.

### Monster Cards

Monster cards show fast-reference information such as:
- HP and temp HP
- AC
- Speed
- Conditions
- Initiative

Clicking a monster's name opens the full monster card, which includes:
- Ability scores
- Saves
- Skills
- Senses
- Resistances and immunities
- Actions
- Reactions
- Legendary actions

### Player Cards

Player cards give the DM a quick view of:
- Name
- HP
- AC
- Passive values
- Spell save
- Conditions

Use `Request Player Stats` when you want updated information from connected player sheets.

### Save and Load Encounters

Use `Save Encounter` to store the current tracker setup for later.

Use `Load Encounter` to restore a saved encounter.

This is helpful for:
- Session prep
- Recurring fights
- Boss encounters
- Dungeon rooms you want ready ahead of time

## Combat Tools In The Tracker

The initiative area also includes additional scene-management tools.

### Timers And Pressure Tools

Depending on your setup, the tracker may include:
- A consequence clock
- A countdown timer
- A fear tracker

These are useful for:
- Timed events
- Environmental pressure
- Failing forward mechanics
- Homebrew table systems

## DM Tables

The DM Tables tab works as a quick-reference and utility section.

It includes tools such as:
- Conditions reference
- Effects table
- Schools of magic
- Travel pace
- Travel service costs
- NPC reference content
- Jump calculator
- Shops
- Random table style utilities

Many tables are editable, so this tab can also double as lightweight session prep.

### Shops

The shop tools are useful for:
- Pulling up prebuilt shops
- Creating custom shop lists
- Reusing item data during play

### Jump Calculator

Use the jump calculator when you want a quick answer for:
- Running jumps
- Standing jumps
- Reach calculations
- Height-based movement rulings

## Checklists

The Checklists tab is a simple task tracker.

Use it for:
- Session prep lists
- NPC reminders
- Plot hooks
- Loot reminders
- In-session objectives

Basic workflow:
1. Add a task.
2. Check it off when complete.
3. Remove it when you no longer need it.

## Spell List

The DM Spell List is a quick spell reference.

Use it to:
- Search for spells
- Read spell descriptions
- Review casting details
- Check components, range, duration, and level

This is especially useful for:
- Monster casters
- NPC spellcasters
- Homebrew spell use during sessions

## Notes

The Notes tab gives you grouped note containers inside the DM Screen.

Use it for:
- Session notes
- NPC reminders
- Plot threads
- Short lore notes
- Travel or dungeon reminders

Best practice:
- Use built-in notes for active session material
- Use an outside note tool or Google Docs for large permanent archives

## Google Docs

The Google Docs tab lets you embed a Google Doc into the DM Screen.

### How To Use It

1. Copy the Google Doc URL.
2. Paste it into the Google Docs field.
3. Load the document.
4. Read it inside the embedded view.

This is best for:
- Session outlines
- Campaign planning documents
- Large lore files
- Notes that are too long for the built-in notes area

## Settings

The settings dropdown controls several global preferences.

### Language

You can switch between supported languages and keep the interface aligned with your preference.

### Theme

You can:
- Use a preset theme
- Use a custom theme
- Adjust primary and secondary colors

### Exhaustion Homebrew

This setting enables the custom exhaustion behavior used by the toolset.

### D&D Version Toggle

Use the version buttons to control whether the interface emphasizes:
- 2014 rules
- 2024 rules
- Both

## Homebrew Creator

The Homebrew area on the DM side is for creating and managing custom content.

Depending on the content type, you can work with:
- Monsters
- Spells
- Items
- Shops

You can also:
- Edit existing homebrew
- Delete unused homebrew
- Watch storage usage for local and global storage

## Suggested DM Workflow

A good DM workflow looks like this:

1. Set language, theme, and rules version.
2. Prepare or load any custom content you need.
3. Save encounters before the session.
4. Use DM Tables and Notes during play.
5. Run combat from the Initiative Tracker.
6. Keep larger campaign documents in Google Docs if needed.

## Tips

- Request player stats when you want the freshest synced player information.
- Save encounters before a session if you expect to reuse them.
- Keep built-in notes concise for best results.
- Use Google Docs for long-term campaign material.
