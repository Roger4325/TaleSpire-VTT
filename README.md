# TaleSpire 5e Toolset

TaleSpire 5e Toolset is a web-view symbiote for TaleSpire that gives Dungeon Masters and players a connected set of tools for running D&D-style games inside the board.

The app routes by mode:
- GM mode opens the DM Screen
- Player mode opens the Player Character Sheet

## What The Toolset Includes

- DM combat tracking with initiative, conditions, encounter saving, and quick monster access
- Player character sheets with editable stats, actions, spells, inventory, traits, notes, and initiative view
- Character import workflows, including supported D&D Beyond conversion
- Custom spells, items, monsters, and shops
- Shared storage for saved content
- Theme, language, and rules-version settings

## Notes

- Built-in notes are best used for active session notes and quick references, not permanent long-form campaign archives.
- Google Docs integration is useful when your longer notes live outside the symbiote.
- Some features interact with each other. Conditions, traits, equipment, and action settings can all affect what appears on the player sheet.
